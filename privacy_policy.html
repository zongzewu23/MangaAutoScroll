Privacy Policy

Manga Auto Scroll does not collect, store, or share any personal data.

Permissions and Functionality

This extension does not automatically inject scripts into webpages. Scripts are only injected when the user explicitly clicks the "Start" button in the extension popup. This ensures that the extension does not run in the background or modify any webpage content without user interaction.

Data Usage

Manga Auto Scroll does not track user activity, collect browsing data, or send any information to external servers. The extension runs entirely on the user's local browser.
